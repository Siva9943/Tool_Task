Help & User Instructions

Everything you need to know to upload, validate, and manage product data.

User Instructions
Uploading products
  1. On the Dashboard, click Upload and download the template file if you don't already have it.
  2. Fill in your product rows without changing the column headers or their order.
  3. Save the file as .xlsx, .xls, or .csv and upload it.
  4. Rows that pass validation are added to your product list immediately.
  5. Rows that fail appear on the Invalid Data page with the exact reason for each failure. Download, correct, and re-upload them — a new upload replaces the previous invalid-row list.
Column rules

Columns marked with * are required; every other column is optional.

Column Type Rules
Product code* Text Required. Must start with "P" followed by 4 or more digits (e.g. P0001). Max 20 characters.
Product name* Text Required. Must start with a letter; letters, numbers, spaces and - ( ) . / allowed. Max 50 characters.
Description Text Optional. Max 150 characters.
Item category Text Optional. Max 50 characters.
Cost price Decimal Optional. Must be 0 or greater, and not exceed ₹1,00,00,000.
Selling price* Decimal Required. Must be 0 or greater.
Quantity* Whole number Required. Must be between 0 and 5000.
Expire date(YYYY-MM-DD) Date Optional. Format YYYY-MM-DD. Cannot be in the past.

Product Code and Product Name must also be unique — both within the uploaded file and against products already in the database.

Help & FAQ

The file's column headers didn't match the template exactly — a column is missing, extra, renamed, or out of order. Re-download the template and copy your data into it without altering the header row.

Each row is validated independently against the column rules above. Visit the Invalid Data page to see the exact reason for every failed row, fix the values, and re-upload.

On the Dashboard, each product card has Update and Delete buttons. Update opens a form to edit the product's details; Delete asks for confirmation before removing it.
