File format, required columns and limitations
Accepted File Format
  • File types: .xlsx, .xls   only
  • First row must contain the exact column headers, in the exact order, shown below
  • Columns marked * are mandatory
  • Do not add, remove, rename or reorder columns
  • Use the Download Template file as a starting point
Notes
  • Rows with duplicate Product Code / Name (within the file or already in the database) are rejected
  • Rejected rows are listed on the Invalid Data page with the reason for failure
  • Valid rows are imported even if some other rows in the same file fail
Column Rules & Limitations
Column Required Rules / Limitations
Product code* Must Starts with P followed by 5+ digits (e.g. P1001); max 20 characters; must be unique (not already used in file or database)
Product name* Must Must start with a letter; letters, numbers, spaces and - ( ) . only; max 50 characters; must be unique
Description Optional Max 150 characters
Item category Optional Max 50 characters
Cost price Optional Decimal number; ₹0 – ₹1,00,00,000
Selling price* Must Decimal number; must be ≥ 0
Quantity* Must Whole number only; 0 – 5000
Expire date(YYYY-MM-DD) Optional Format YYYY-MM-DD; cannot be a past date
